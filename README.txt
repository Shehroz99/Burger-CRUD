READ ME! 

This is a fullstack CRUD web application built on React and .NET

You need to have dotnet and node.js installed on your computer. 
https://dotnet.microsoft.com/en-us/download 
https://nodejs.org/en/download/package-manager

Download and unzip the file

Using the Command Line set your path to the burgerAPI folder 
Enter the following command: 

“ dotnet run “ 

Voila! Your web API is now being hosted by you

Using the Command Line set your path to the Burger23 folder
Enter the following command: 

“ npm install “

Followed by 

“npm run dev” 

Then press “O” in the Command line to access the web page. 

Your fullstack burger CRUD application has now been set up!
